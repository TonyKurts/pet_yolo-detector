# Подсчет посетителей магазина

Одностраничное FastAPI-приложение принимает фотографию, запускает YOLOv11 в
фоновой задаче, считает только людей и сохраняет изображение с bounding boxes.
Результаты хранятся в SQLite и доступны в интерфейсе и CSV.

## Запуск

Требуется Python 3.10 или новее.

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Приложение откроется по адресу <http://127.0.0.1:8000>. При первом запуске
инференса Ultralytics автоматически загрузит веса `yolo11n.pt`, если файла еще
нет локально. Запускайте приложение с одним Uvicorn worker: глобальная очередь
обработки и singleton модели рассчитаны на один серверный процесс.

## API

- `POST /api/upload` — загрузить PNG/JPEG и запустить фоновую обработку.
- `GET /api/status?task_id=...` — получить статус задачи или последней задачи.
- `GET /api/history` — получить историю в порядке от новых записей к старым.
- `GET /api/export` — скачать полный снимок таблицы в `export.csv`.
- `/storage/...` — скачать обработанное изображение.

Статусы: `0` - new, `1` - process, `2` - complete, `3` - error.

Класс человека фильтруется на этапе инференса обязательным аргументом
`classes=[0]` в `model.predict(...)`.
